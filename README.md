## 📌 简介

玩文字游戏时，满屏字幕却没有语音，沉浸感大打折扣。

**GameVoiceAssistant** 就是为解决这个问题而生。它通过识别游戏字幕和角色，自动匹配并播放对应的语音——可以是TTS语音/AI合成，也可以是真人录制，后期。

✅ **完全免费 · 绿色无广告 · 持续更新**

---

## ✨ 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 🎮 通用适配 | 支持任何文字类游戏（AVG、视觉小说、互动故事） |
| 🔍 自动识别 | 识别游戏中的字幕文本和当前说话角色 |
| 🔊 语音匹配 | 根据文字+角色自动匹配并播放语音 |
| 🎙️ 多种音源 | 支持AI合成语音 / 真人录制语音包 |
